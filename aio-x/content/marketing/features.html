Características y Beneficios

Multiplataforma
Aplicaciónes Web Progresivas(PWA)

Aproveche las características modernas de la plataforma web que brindan una experiencia similar a la aplicación. Alto rendimiento, soporte fuera de línea e instalación de cero pasos

Aplicaciónes Nativas

Desarrolle estratégicamente aplicaciones nativas móviles con Ionic y NativeScript

Aplicaciónes de escritorio

Cree aplicaciones de escritorio para Mac, Windows y Linux con métodos angulares y acceda a las API nativas del sistema operativo.

Velocidad y rendimiento
Generación de código

Angular traduce plantillas en código altamente optimizado para las máquinas virtuales JavaScript de hoy, brindando todos los beneficios del código escrito a mano para la productividad del marco de desarollo(framework).

Universal

Puede mostrar la primera vista de su aplicación en un servidor como Node.js®, .NET, PHP y otros servidores para la representación casi instantánea en solo HTML y CSS. También acondiciona el camino para optimizar el SEO de su sitio.

División de código

La aplicación Angular se cargará inmediatamente con el nuevo enrutador de componentes. El nuevo enrutador de componentes proporciona división automática de código para que los usuarios solo carguen el código necesario para representar la vista solicitada.

Productividad
Plantillas

Puede crear rápidamente vistas de Interfaz de Usuario(UI) utilizando la sintaxis de plantillas simple pero potente.

Angular CLI

Herramientas de línea de comandos: inicie compilaciones rápidas, agregue componentes y pruebas e implemente fácilmente.

IDE´s

Obtenga autocompletacion de código inteligentemente, errores instantáneos y otros comentarios con editores e IDE populares.

Historia de desarrollo completa
Puesta a Prueba(Testing)

Las pruebas unitarias de Karma te dirán si algo se rompió cada vez que guardas. Y Protractor hace que sus pruebas de escenarios se ejecuten más rápido y de manera más consistente.

Animación

Con la API intuitiva de Angular, puede crear coreografías complejas de alto rendimiento y líneas de tiempo animadas con poco código.

Accesibilidad

Cree aplicaciones accesibles con componentes habilitados para ARIA, guías para desarrolladores e infraestructura de prueba a11y integrada.
